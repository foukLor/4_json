# 4_json


This script is useful for pretty print json:
ordinary:
```json
{ "foo": "first", "bar": "second" }
```

after using script:

```json
{
    "foo": "first",
    "bar": "second"
}
```




For running please use python3.5

## Usage
```sh
python pprint_json.py path_to_file
```

The file is a json document.
